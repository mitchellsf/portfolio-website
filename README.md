# Quarto portfolio scaffold

## 1) Prereqs
- Install Quarto: https://quarto.org
- Optional: Python if you want to render notebooks or build the game bundle.

## 2) Preview locally
From this folder:

    quarto preview

## 3) Render
    quarto render

Output goes to `_site/`.

## 4) Publish to GitHub Pages
Quarto can publish directly:

    quarto publish gh-pages

(Your repo needs to be on GitHub first.)
